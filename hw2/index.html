Part 2: Bezier surfaces with separable 1D de Casteljau

The extenstion to the surface is just doing the same things twice. In the first pass, we apply the de Casteljau algorithm along each row of control points using parameter u, producing intermediate points. Then, in the second pass, we follow the parameter v to producing the final surface points. This logic is implemented in the "evaluate" function. The evaluateStep function performs one interpolation step in 3D space, similar to the Part 1 but using Vector3D. The evaluate1D is an intermediate function that applies the de Casteljau algorithm to a single dimension (either u or v) using the evaluateStep function.
